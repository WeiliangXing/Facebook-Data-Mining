

1)Files:
CLASSIFIER.py : main program which will be run
data folder: all initial data provided by the project.
result.txt: the output file for different smooth parameter values. 

2) To run
To run it, do:
python3.4 CLASSIFIER.py

When it shows “Done.” in the terminal, the program finishes.

2)Note
Please define your own list for alpha values to get results.
In the program, the default list of alpha for experiments are [200000, 1000000];
All results will be appended to result.txt;
All results will be drawn in result.png.
